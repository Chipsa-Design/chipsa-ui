# Chipsa UI

UI библиотека для современных веб-приложений.

## Структура проекта

Монорепозиторий, состоящий из следующих пакетов:

- **@chipsa-ui/core** — базовый пакет с основной функциональностью
- **@chipsa-ui/react** — React-обертка для использования в React-приложениях
- **@chipsa-ui/astro** — Astro-обертка для использования в Astro-проектах

## Установка

```bash
pnpm install
```

## Разработка

```bash
# Дока dev-режим
cd apps/www
pnpm install
pnpm dev
```

```bash
# Сборка всех пакетов
pnpm build
```
