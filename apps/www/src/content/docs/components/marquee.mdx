## Описание

Marquee — это компонент для создания бесконечной горизонтальной прокрутки элементов. Компонент автоматически дублирует контент для создания эффекта непрерывной прокрутки и поддерживает настройку скорости, направления и других параметров анимации.

## Основные возможности

- Бесконечная горизонтальная прокрутка
- Настраиваемая скорость и направление движения
- Автоматическая пауза при наведении (опционально)
- Поддержка перетаскивания (drag & drop)
- Автоматическая остановка анимации вне вьюпорта
- Автоматическое заполнение контента при недостаточной ширине

## API

### Props

#### `items` (required)

`unknown[]` — массив элементов для отображения в marquee.

#### Остальные HTML атрибуты

Компонент принимает все стандартные HTML атрибуты элемента `<div>`.

### Слоты

#### Default slot

Слот для рендеринга каждого элемента из массива `items`. Получает элемент массива в качестве параметра.

## Использование

### Базовое использование

```astro
---
import Marquee from '@astro/components/Marquee';

const items = ['Элемент 1', 'Элемент 2', 'Элемент 3'];
---

<Marquee items={items}>
    {(item) => <div>{item}</div>}
</Marquee>
```

### С настройкой скорости и направления

Компонент автоматически инициализируется с настройками по умолчанию (speed: 30, direction: 'left'). Для кастомизации можно изменить скрипт в компоненте или добавить data-атрибуты для передачи параметров.

## Настройки JavaScript

Компонент использует класс `Marquee` из `lib/Marquee.ts` со следующими опциями:

- `speed` — скорость прокрутки в пикселях в секунду (по умолчанию: 30)
- `direction` — направление прокрутки: `'left'` или `'right'` (по умолчанию: `'left'`)
- `pauseOnHover` — пауза при наведении мыши (по умолчанию: `false`)
- `autoFill` — автоматическое заполнение контента (по умолчанию: `true`)
- `delay` — задержка перед началом анимации в секундах (по умолчанию: 0)
- `draggable` — возможность перетаскивания (по умолчанию: `false`)

## CSS классы

- `.marquee` — основной контейнер
- `.marquee--draggable` — добавляется при включенном режиме перетаскивания
- `.marquee__element` — элемент с анимацией
- `.marquee__container` — контейнер с элементами
- `.marquee__item-wrapper` — обёртка для каждого элемента

## Поведение

Компонент автоматически:

- Создаёт дубликаты контента для бесконечной прокрутки
- Останавливает анимацию когда элемент вне вьюпорта (для оптимизации производительности)
- Обрабатывает изменение размера окна и пересчитывает позицию
- Поддерживает перетаскивание при включенной опции `draggable`
