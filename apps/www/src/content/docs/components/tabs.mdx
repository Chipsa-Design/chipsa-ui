## Описание

Tabs — это компонент вкладок с поддержкой клавиатурной навигации и полной accessibility. Компонент автоматически управляет состоянием вкладок, добавляет необходимые ARIA-атрибуты и обеспечивает навигацию с клавиатуры.

## Основные возможности

- Клавиатурная навигация (стрелки, Home, End, Enter, Space)
- Полная поддержка accessibility (ARIA-атрибуты)
- Автоматическое управление состоянием активной вкладки
- Гибкая структура через data-атрибуты
- Поддержка всех стандартных HTML атрибутов

## API

### Props

#### `id` (required)

`string` — уникальный идентификатор компонента вкладок. Используется для группировки элементов вкладок.

#### `class?`

`string` — дополнительные CSS классы для кастомизации стилей.

#### Другие HTML атрибуты

Компонент поддерживает все стандартные HTML атрибуты элемента `<div>`.

## Data-атрибуты

### `data-tabs-toggler`

Устанавливается на элемент (обычно кнопку), который переключает вкладку. Значение должно совпадать с `id` содержимого вкладки.

```html
<button data-tabs-toggler="tab-1">Вкладка 1</button>
```

### `data-tabs-content`

Устанавливается на элемент, который содержит содержимое вкладки. Элемент должен иметь `id`, соответствующий значению `data-tabs-toggler`.

```html
<div id="tab-1" data-tabs-content>Содержимое вкладки</div>
```

### `data-tabs-id`

Автоматически устанавливается компонентом для группировки элементов вкладок. Обычно не требуется устанавливать вручную.

## Клавиатурная навигация

Компонент поддерживает полную клавиатурную навигацию:

- **←/↑** — переключение на предыдущую вкладку
- **→/↓** — переключение на следующую вкладку
- **Home** — переход к первой вкладке
- **End** — переход к последней вкладке
- **Enter/Space** — активация текущей вкладки

## Accessibility

Компонент автоматически добавляет необходимые ARIA-атрибуты:

- `role="tablist"` для контейнера вкладок
- `role="tab"` для переключателей вкладок
- `role="tabpanel"` для содержимого вкладок
- `aria-selected` для указания активной вкладки
- `aria-hidden` для скрытого содержимого
- `aria-controls` и `aria-labelledby` для связывания элементов

## CSS классы

Компонент использует следующие классы:

- `tabs` — базовый класс контейнера
- `js-tabs` — класс для инициализации JavaScript функциональности
- `is-active` — класс для активной вкладки (добавляется вручную на первую вкладку)
