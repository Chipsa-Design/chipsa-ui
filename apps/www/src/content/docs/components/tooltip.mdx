## Описание

Tooltip — это компонент всплывающей подсказки с автоматическим позиционированием и плавными анимациями. Компонент автоматически обрабатывает показ/скрытие подсказки при наведении, фокусе или клике на основном элементе.

## Основные возможности

- Автоматическое позиционирование с предотвращением выхода за границы экрана
- Плавные анимации появления и исчезновения
- Поддержка hover, focus и click событий
- Автоматическое переворачивание при нехватке места
- Кастомизация через CSS-переменные

## API

### Props

Компонент поддерживает все стандартные HTML атрибуты элемента `<div>`.

#### `class?`

`string` — дополнительные CSS классы для кастомизации стилей.

### Слоты

#### `default` (required)

Основной контент, на который будет навешиваться подсказка. Может быть любым интерактивным элементом (кнопка, ссылка и т.д.).

#### `tooltip` (required)

Содержимое подсказки. Отображается при наведении, фокусе или клике на основном элементе.

## Использование

### Базовое использование

```astro
---
import Tooltip from '@astro/components/Tooltip/Tooltip.astro';
---

<Tooltip>
    <button>Наведите на меня</button>
    <div slot="tooltip">Это всплывающая подсказка</div>
</Tooltip>
```

### С ссылкой

```astro
---
import Tooltip from '@astro/components/Tooltip/Tooltip.astro';
---

<Tooltip>
    <a href="/docs">Документация</a>
    <div slot="tooltip">Перейти к документации</div>
</Tooltip>
```

### Кастомизация стилей

```astro
---
import Tooltip from '@astro/components/Tooltip/Tooltip.astro';
---

<Tooltip class="my-tooltip">
    <button>Кастомная подсказка</button>
    <div slot="tooltip">Подсказка с кастомными стилями</div>
</Tooltip>

<style>
    .my-tooltip {
        --bg-color: #ff0000;
        --border-color: #cc0000;
    }
</style>
```

## События

Компонент автоматически обрабатывает следующие события:

- **Показ**: hover, focus, click на основном элементе
- **Скрытие**: mouseleave, blur с задержкой 100ms

## CSS-переменные

Компонент использует следующие CSS-переменные для кастомизации:

- `--bg-color` — цвет фона подсказки (по умолчанию: `#4e4e4e`)
- `--border-color` — цвет границы (по умолчанию: `#484848`)
- `--shift-x` — автоматически устанавливаемое смещение для стрелки (устанавливается автоматически)
